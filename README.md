# nonebot-plugin-Jinshan-Railway

###  ✨“下一站，终点站：莘庄，可换乘5号线，可出站换乘金山铁路”✨

#### 这是基于Nonebot2的很实用的金山铁路机器人，可以查询金山铁路列车车次详细信息、车站当日列车时刻表，将车站大屏公交化显示

**（该项目还在开发中）**

</div>

## Required 🔨
- Python >= 3.8

## Getting Started 🚀

将本repo中的nonebot-plugin-railwaytools文件夹clone到机器人的插件目录中，即可使用。

```sh
git clone https://github.com/leaf2006/nonebot-plugin-railwaytools.git
```

## Features ✨

- /小火车 [金山铁路列车车次] 或 /xhc [金山铁路列车车次] ： 通过车次查询该列车详细信息

```
example:
🤵：/小火车 S1633
🤖：
🚝S1633次列车：
类型：站站停
------------------------------ 
莘庄：--到，21:49开 
春申：21:55到，21:56开 
新桥：22:04到，22:05开 
车墩：22:12到，22:13开 
叶榭：22:20到，22:21开 
亭林：22:27到，22:28开 
金山园区：22:34到，22:35开 
金山卫：22:45到，--开 

------------------------------ 
数据更新时间：2025-10-11
```

- /车站 [金山铁路车站名称] 或 /cz [金山铁路车站名称] ： 通过车站名称查看该车站大屏 **（显示在列表中的列车的发车时间都是晚于你在查询时的时间的，所以不用担心赶不上列车）**

```
example:
🤵：/车站 [叶榭]
🤖：
🚉叶榭站列车信息:
------------------------------ 
S1207，下行（金山卫方向）：21:35到，21:36开
S1633，下行（金山卫方向）：22:20到，22:21开
S1211，下行（金山卫方向）：22:33到，22:34开
S1210，上行（市区方向）：21:35到，21:37开
------------------------------ 
数据更新时间：2025-10-11
```

## Alert ⚠️

- 这个插件所使用的信息是真实存在的，但是这个json文件不是现成的。是我使用金山铁路微信公众号里的时刻表图片，丢给Google Gemini并添加了部分提示词生成的。虽然说这么做时效性上肯定是差了点，但是鉴于金山铁路只要不调图，时刻表肯定不怎么会变。等到调图的时候我再把数据更新一下便好了

## 数据来源

- 金山铁路微信公众号
